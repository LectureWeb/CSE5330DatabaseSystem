Course Content:

This is a first course in database systems. The prerequisites are knowledge of programming and data structures (CSE 2320 or equivalent). The course will provide a good understanding of the database theory, the relational model and its characteristics. Relational algebra, principles and SQL database language and some programming techniques will also be part of this course. The conceptual database design using the Entity-Relationship model and Extended Entity-Relationship modeling, and mapping to relational model and the file and index structures and search methods for database storage will also be covered in class. Finally the course content covers the relational database theory the functional dependencies, normalization and the main principles of database security. We will cover the following chapters from the textbook (7th edition chapters in red). Parts or all of items 8 through 10 (below) may be omitted if time does not permit to cover all the topics.

1. Chapters 1, 2: Introduction to database concepts and architecture.(Chapter 1, 2)
2. Chapter 3: The relational data model (Chapter 5).
3. Chapters 4, 5: The SQL database language. (Chapter 6, 7)
4. Chapter 6 (Sections 6.1-6.5): The relational algebra(Chapter 8)
